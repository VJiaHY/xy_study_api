基础微服务平台
===============
版本:v1.0.0

发布日期：2022-04-27 作者:JiaHY


## 后端技术架构
- 基础框架：Spring Boot 

- 网关框架： Spring Cloud GatWay

- 服务注册发现： 阿里巴巴 Nacos

- 持久层框架：Mybatis-plus 

- 安全框架：Spring Security ，Jwt 

- 数据库连接池：阿里巴巴Druid 

- 缓存框架：redis

- 其他：poi，oshi，lombok（简化代码）等。

## 开发环境

- 语言：Java 8

- IDE(JAVA)： Eclipse安装lombok插件 或者 IDEA

- 依赖管理：Maven

- 数据库：MySQL 8.5

- 缓存：Redis
  
- 服务注册发现：Nacos
